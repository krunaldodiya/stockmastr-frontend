For Android:

Manually change it in android/app/src/main/java/com/PROJECT_NAME/MainActivity.java:
package com.ssx.socialstock;

In android/app/src/main/java/com/PROJECT_NAME/MainApplication.java:
package com.ssx.socialstock;

In android/app/src/main/AndroidManifest.xml:
package="com.ssx.socialstock"

In android/app/build.gradle:
applicationId "com.ssx.socialstock"

Gradle' cleaning in the end (in /android folder):
./gradlew clean


For iOS:

Change the Bundle Id in Xcode.

<PROJECT_NAME> > General > Bundle Identifier
com.ssx.socialstock
<PROJECT_NAME-tvOS> > General > Bundle Identifier
com.ssx.socialstock-tvOS
